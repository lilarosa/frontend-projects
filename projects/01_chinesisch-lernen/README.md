# Chinese Learning Website

## Project Overview

Chinese Learning Website is a child-friendly learning portal for Chinese language practice.  
It combines character learning, pinyin support, poem practice, small games and attendance/check-in elements into one playful learning experience.

## Target Users

- Children who are starting to learn Chinese
- Parents or teachers who want a simple practice website
- Learners who benefit from visual repetition and small interactive tasks

## Core Features

- Chinese character review for the current week
- Pinyin and picture-based word practice
- Poem learning section
- Small interactive learning game
- Calendar/check-in area for learning habits
- Navigation for homework, feedback and communication pages

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript
- Static GitHub Pages deployment
