# Hungary Survival Guide

## Project Overview

Hungary Survival Guide is a practical web app for newcomers, students and expats in Hungary.
It collects everyday survival information into one mobile-friendly interface: language help, healthcare, safety, lifestyle, culture and leisure.

## Target Users

- Expats and newcomers in Hungary
- International students
- Travelers staying longer in Budapest or other Hungarian cities
- People who need quick phrase support in everyday situations

## Core Features

- Scenario-based Hungarian phrase practice
- Essential tool words with horizontally scrollable reveal cards
- Healthcare, safety, lifestyle, culture and leisure modules
- Practical local information for everyday life
- PWA manifest for installable app behavior
- Local state support through localStorage in selected flows

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript
- localStorage
- PWA manifest
- GitHub Pages deployment
