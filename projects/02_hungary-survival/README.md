# Hungary Survival (Web App)

## DE: Projektueberblick
Eine Web-App fuer Expats in Ungarn mit praxisnahen Informationen zu Alltag, Sicherheit und Orientierung in Budapest.

## DE: Funktionen
- Interaktive Inhalte fuer Sprache, Freizeit, Gesundheit und Sicherheit
- Mehrere thematische Seiten (`language.html`, `healthcare.html`, `safety.html` usw.)
- Lokale Speicherung ueber `localStorage`

## DE: Technik
- HTML, CSS, Vanilla JavaScript
- Deployment ueber GitHub Pages
- PWA-Datei: `manifest.json`

## DE: Demo
- https://lilarosa.github.io/hungary-survival/
