Project notes

Built as a Flutter + backend learning project

The original app lives locally as a Flutter project named smart_outfit_flutter. This GitHub Pages version is a static showcase, because the full app expects local API services for clothes, image upload, and recommendations.

  • Flutter UI for clothes and recommendations
  • Spring-style backend model for wardrobe data
  • Image upload workflow and outfit matching logic
