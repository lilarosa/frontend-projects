# CareU

## Project Overview

CareU is an elder-care companion prototype.  
It focuses on simple daily support: reminders, quick call actions, reading mode, speech support and a light memory game.

## Target Users

- Older adults who need a simple daily routine interface
- Family members who want a gentle support tool
- Care scenarios where clarity, large touch targets and low cognitive load matter

## Core Features

- Current time and daily greeting
- Medicine, meal and task reminders
- Quick call buttons for friend, family and emergency support
- Daily reading section
- Speech synthesis support for reading content aloud
- Memory-match brain game
- PWA/service worker structure

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript
- Web Speech API
- PWA/service worker
