# AP1 Study App

## Project Overview

AP1 Study App is a focused study assistant for Fachinformatiker AP1 exam preparation.
It organizes exam topics, weekly planning, checklist workflows and quick quiz practice into one compact PWA-style interface.

## Target Users

- Fachinformatiker trainees preparing for AP1
- Learners who need a structured weekly study plan
- Students who want quick revision blocks for SQL, Java, UML, testing, networks and Linux basics

## Core Features

- AP1 topic modules
- Weekly study planning area
- Checklist-based progress workflow
- Quick quiz section
- Responsive single-page layout
- PWA-ready structure

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript
- PWA manifest/service worker structure
- GitHub Pages deployment
